Detailed Description

template<typename T, size_t N>
class NetCDFField< T, N >

A class representing a field of data read from a NetCDF file.

Template Parameters
TThe data type of the field (e.g., float, double).
NThe number of components per data point (e.g., 1 for scalar fields, 3 for vector fields).
Note
The current implementation reads 3D data for each variable separately. Future improvements may involve using read1DVariablesAtTime for better compatibility with configurable indexing.
Exceptions
std::runtime_errorif reading from the NetCDF file fails.
