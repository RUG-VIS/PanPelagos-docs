Detailed Description

template<typename T>
class Vec3< T >

A 3D vector class template that extends the generic Vec class.

This class provides additional functionality specific to 3D vectors, such as cross product and Rodrigues rotation.

Template Parameters
TThe type of the vector components (e.g., float, double).
