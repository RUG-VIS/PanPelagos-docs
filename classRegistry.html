std::unique_ptr< BaseParticleSet > Registry::loadParticleSet ( const std::string  path)

Loads a particle-set.

While paritclesets themselves are not pluginified, the particle I/O (reader and writer) are. This function loads a particle set with the I/O set according to the configuration.

Parameters
paththe path (including filename) to the particle file.
Returns
shared_ptr to an particleset (either list-based or array-based, depending on configuration), with both the reader and writer as plugins. This set contains the particle information located inside path.
