Detailed Description

template<typename T, std::size_t N>
class Vec< T, N >

A templated N-dimensional vector class for mathematical operations.

This class provides a generic implementation of a vector in N-dimensional space, where the dimension is specified at compile time. It supports various mathematical operations such as addition, subtraction, scalar multiplication, dot product, and more. The operations are optimized for performance and can be used in a wide range of applications.

Template Parameters
TThe type of the vector elements (e.g., float, double, int).
NThe dimension of the vector (must be greater than 0).
