void BaseParticleIO::addVariablesToIO ( std::vector< std::vector< std::string >>  varNames)

Adds scalar or vec3 variables to the IO system based on variable names.

  • If the variable name has size 2: scalar.
  • If the variable name has size 4: vector.
  • If the descriptor has any other size: throws a runtime error.
Parameters
varNamesA std::vector of variable names
Exceptions
std::runtime_errorif a descriptor does not have the correct size (2 or 4).
