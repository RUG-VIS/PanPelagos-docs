Detailed Description

template<typename T, size_t N>
class FieldTimeBuffer< T, N >

A specialized TimeBuffer for managing and caching Field objects.

This class is designed to handle the loading and caching of Field objects from NetCDF files or Python wrappers, depending on the build configuration. It uses a TimeBuffer to manage the temporal data and provides an efficient mechanism to load and retrieve fields based on time keys.

Template Parameters
TThe data type of the field (e.g., float, double).
NThe number of variables to be managed by the buffer.
