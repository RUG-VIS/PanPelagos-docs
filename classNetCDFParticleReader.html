Detailed Description

template<typename T, typename ParticleType>
class NetCDFParticleReader< T, ParticleType >

Implements particle reading for the NetCDF (.nc) file extension, as plugin.

Template Parameters
Tprimitive data type used for particle values (e.g. floats for position).
ParticleTypewhich ParticleVariant implementation to use. Must be either of CppParticle or FastCppParticle in current implementation.

Implements particle reading for the NetCDF (.nc) file extension, as plugin. This version uses the PyParticle ParticleVariant, and therefore functions (exclusively) with ShedSkin.
